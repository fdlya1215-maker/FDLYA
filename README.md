我使用 AI 辅助开发构建了一个浏览器端图片压缩工具。该工具解决的核心痛点是：用户在处理证件照、报名照片等场景时，现有本地压缩工具（如画图工具、微信传输等）通常只能将图片压缩到约 46KB 左右，而许多专业考试报名系统（如注册会计师考试、公务员考试、各类职业资格认证）对证件照文件大小有严格限制，通常要求控制在 20KB 甚至 10KB 以内。普通用户很难通过常规手段达到这一要求，往往需要反复尝试或寻找专业软件，体验极差。
该工具是一个纯前端单页 HTML 应用，基于 Canvas API 在浏览器本地完成所有压缩处理。核心逻辑流包括：（1）拖拽上传，支持 JPG/PNG/WebP 格式；（2）通过滑块实时调节输出质量（5%-100%），可精细控制文件大小，轻松压缩至 20KB 以内，满足各类报名系统的严格要求；（3）支持 JPG、PNG、WebP 三种格式互转，PNG 转 JPG 可进一步大幅降低文件体积；（4）可选调整图片尺寸（如将证件照调整为报名要求的像素规格），支持锁定宽高比等比缩放；（5）原图与压缩图左右对比预览，实时显示文件大小、压缩率、节省空间等数据；（6）一键下载压缩后的文件。
